# CentOS SSH

> https://hub.docker.com/r/anshengme/centos-ssh/

## Building & Running

```bash
$ docker build -t centos-ssh .
$ docker run -d -p 22:22 centos-ssh
```

## Pull & Running

```bash
$ docker pull anshengme/centos-ssh
$ docker run -d -p 22:22 anshengme/centos-ssh
```
